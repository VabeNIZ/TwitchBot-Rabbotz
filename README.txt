Перед запуском бота обязательно установить:
pip и pycharm

pip модуль для виндовс, который позволяет быстро и удобно устанавливать библиотеки для питона посредством коротких строк в командой строке

pycharm среда, в которой и нужно будет запускать код



В консоли прописать:
pip install yandex_speech
pip install gTTS
pip install pygame


Без этих библиотек не будет работать озвучка сообщений, а конкретно: 

yandex_speech отвечает за спич кит от яндекса, то есть яндексовские голоса доступны не буду, придется указать в конфиге в соостветствующем поле "google"

gTTS тоже самое, что и выше, но наоборот - не будет доступен гугл, а только яндекс, который нужно указать в конфиге и соответствующие с ним параметры

pygame - библиотека, позволяющая проигрывать mp3 файлы (на самом деле она в основном используется для создания игор, но нам нужна только для звуков)



log.txt - собственно файл лога. прописываются все сообщения и выводятся ошибки для дальнейшей отладки
makebuild.bat - батник для создания билдов. то что я использовал, чтобы скинуть тебе екзешник. тебе он уже не нужен.
